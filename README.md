# mediapipe-vr

An implementantion of google tracking for vrchat
things needed:
 - Phone (Cardboard)/Oculus Go/Pimax 4K
 - PS Move/PS Navi/Wii remote
 - VR ready PC
 - ~2 or more cameras with medium quality (PS3 Eye)
